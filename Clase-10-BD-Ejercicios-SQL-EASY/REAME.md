# Clase 10: Ejercicios de Bases de Datos SQL (Fácil)

## Descripción General

En esta clase se presentan una serie de ejercicios prácticos con un nivel de dificultad moderado para familiarizarse con los conceptos fundamentales de SQL. Los ejercicios están diseñados para ayudar a entender y aplicar consultas SQL básicas de una manera más accesible.

## Requisitos Previos
Para seguir estos ejercicios, se recomienda tener conocimiento básico de los siguientes temas:

- Consultas SELECT.
- Filtros con WHERE.
- Uso de operadores lógicos (AND, OR).
- Creación de tablas y manejo de datos.

## Instrucciones
1. Clona este repositorio en tu máquina local.
2. Crea una base de datos en tu gestor de bases de datos:
   ```sql
   CREATE DATABASE ejemplo_bd_easy;
   USE ejemplo_bd_easy;
   ```
3. Ejecuta los archivos SQL en tu editor preferido, como DBeaver o MySQL Workbench.

## Ejercicios
Cada archivo `.sql` en esta carpeta contiene consultas prácticas que puedes ejecutar directamente. Están estructurados para que practiques paso a paso las diferentes funcionalidades de SQL.

¡Buena suerte y disfruta aprendiendo SQL!
